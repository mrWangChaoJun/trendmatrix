# 前端开发计划与页面功能清单

## 1. 项目概述

### 1.1 技术栈
- **框架**：Vue 3 + TypeScript
- **构建工具**：Vite
- **状态管理**：Pinia
- **路由**：Vue Router
- **HTTP客户端**：Axios
- **数据可视化**：D3.js
- **样式框架**：Tailwind CSS
- **代码规范**：ESLint + Prettier

### 1.2 现有页面结构
- `/` - 仪表盘
- `/signals` - 信号系统
- `/signals/history` - 信号历史
- `/signals/:id` - 信号详情
- `/solana` - Solana 生态分析
- `/solana/project/:id` - 项目分析
- `/solana/nft` - NFT 市场分析
- `/solana/defi` - DeFi 协议分析
- `/settings` - 系统设置
- `/auth/login` - 登录
- `/auth/register` - 注册
- `/*` - 404 页面

## 4. API 集成计划

### 4.1 认证 API
- `POST /api/auth/register` - 用户注册
- `POST /api/auth/login` - 用户登录
- `POST /api/auth/logout` - 用户登出
- `POST /api/auth/refresh` - 刷新令牌
- `GET /api/auth/me` - 获取当前用户信息

### 4.2 订阅 API
- `GET /api/commerce/subscriptions/plans` - 获取订阅计划
- `POST /api/commerce/subscriptions` - 创建订阅
- `GET /api/commerce/subscriptions` - 获取用户订阅
- `PUT /api/commerce/subscriptions/:id` - 更新订阅
- `DELETE /api/commerce/subscriptions/:id` - 取消订阅

### 4.3 支付 API
- `GET /api/commerce/payments/methods` - 获取支付方式
- `POST /api/commerce/payments/methods` - 添加支付方式
- `DELETE /api/commerce/payments/methods/:id` - 删除支付方式
- `POST /api/commerce/payments` - 创建支付
- `GET /api/commerce/payments/history` - 获取支付历史

### 4.4 信号 API
- `GET /api/signals` - 获取信号列表
- `GET /api/signals/:id` - 获取信号详情
- `GET /api/signals/history` - 获取信号历史
- `POST /api/signals/generate` - 生成新信号

### 4.5 Solana API
- `GET /api/solana/ecosystem` - 获取生态系统数据
- `GET /api/solana/projects` - 获取项目列表
- `GET /api/solana/projects/:id` - 获取项目详情
- `GET /api/solana/nft/market` - 获取 NFT 市场数据
- `GET /api/solana/defi/protocols` - 获取 DeFi 协议数据
